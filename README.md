# Sony Headphone Ctl

A small cli tool to control the Sony WF-1000XM4 (Earbuds) 

## Features

### Report

- [ ] Battery
- [ ] Device Info
- [ ] Registered Devices


### Config

- [X] Ambient sound control
- [ ] Equalizer
- [ ] Connection Quality
- [ ] DSEE Extreme
- [ ] Speak-To-Chat
- [ ] Automatic Power Off
- [ ] Change Touch sensor function
- [ ] On device ASC settings
- [ ] BT Multipoint
